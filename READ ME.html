READ ME

  1. Why Use This MVC Framework?

    This MVC can be used to create a list of vendors and the products the vendors sell.

  2. System Requirements

    * PHP, Apache, MySQL, Javascript
    * Internet Browser (Google Chrome)

  3. How To Install This MVC

    If you are using WAMP, do the following:
    * Unzip this folder.
    * Copy the unzipped folder.
    * Paste the unzipped folder in the C:\wamp\www folder.
    * Switch on WAMP.
    * Open an internet browser such as Google Chrome.

  4. How To View Vendor's Information

    1. Make sure PHP, APACHE and MYSQL is switched on.
    2. Open an internet browser and type 127.0.0.1/sheku_vendors_mvc/index.php in the web browser's address bar.
      You should see a page such as the one shown in the picture below.

  5. Add Vendor's New Products

    1. Click on the View Profile link.
    2. Click on the Click Here To Add Products link.
    3. Type the products' information into the form.
    4. Tick the checkboxes of the records to be submitted.
    5. Click on the submit button to add the records.

  6. Features To Be Added In The Future

    • Vendors will be able to sign up for new profiles and log into their accounts.
    • Vendors will be able to add, delete and update their profiles.
    • The products' full description and specifications.
    • Javascript and PHP form validation.
